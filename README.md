# 📚 SmartTracker – AI-Assisted Smart Attendance & Student Engagement System

> Making classroom attendance faster, smarter, and more engaging.

---

## 📖 Overview

SmartTracker is an AI-assisted classroom attendance and student engagement system designed to simplify attendance management for educational institutions.

Traditional attendance consumes valuable classroom time because teachers have to call every student's name manually. SmartTracker replaces this process with a secure one-time verification code generated by the teacher.

Students simply enter the code during class, and their attendance is recorded instantly. The system also provides attendance analytics, student management, and an incentive system that rewards students for consistent participation and good performance.

This project aims to reduce administrative work, improve classroom efficiency, and help educators make data-driven decisions.

---

## ❗ Problem Statement

In many schools and colleges:

- Teachers spend several minutes taking attendance manually.
- Attendance records are difficult to manage.
- Identifying low-attendance students requires extra effort.
- Student performance data is scattered.
- Rewarding consistent students is mostly done manually.

These repetitive tasks reduce valuable teaching time.

---

## 💡 Solution

SmartTracker digitizes attendance using a secure verification code.

### Workflow

1. Teacher logs into the Admin Dashboard.
2. Teacher generates a secure attendance code.
3. Students log into the Student Portal.
4. Students enter the verification code.
5. Attendance is recorded automatically.
6. Attendance reports and analytics are updated instantly.
7. Teachers can reward students with incentives.
